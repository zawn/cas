# OpenID连接协议

您可以使用以下方式配置CAS服务器：

* [OIDC客户端支持](../integration/Delegate-Authentication.html) 登录页面上的链接将身份验证委派给OpenID Connect提供程序。
* [OIDC服务器支持](../installation/OIDC-Authentication.html)，这意味着您可以通过CAS作为OP [OpenID Connect协议](http://openid.net/connect/)
